# 📋 Project Requirements

## Project Name

**Smart Banking Analytics Dashboard**

---

# 🎯 Project Objective

Develop an end-to-end Banking Analytics solution that combines data analysis, business intelligence, and machine learning to analyze customer behavior, banking operations, and predict loan default risk.

---

# 🖥 Software Requirements

- Microsoft Excel 2019 or later
- Microsoft Power BI Desktop
- Python 3.10 or later
- Google Colab or Jupyter Notebook
- DuckDB or SQLite
- Git (Optional)
- GitHub Account (Optional)
